NeuroClaw FAQ

NeuroClaw 需要收费吗？
不需要。NeuroClaw 仓库是公开的，可按照仓库中的许可证使用。你不需要为查看代码或在本地运行平台本身付费；但你自己的计算资源、存储空间以及第三方服务费用仍然需要承担。
NeuroClaw 的能力能兼容其他框架或者 IDE 吗？
NeuroClaw 的核心技能与 OpenClaw、Hermes、Claude Code 等平台兼容。我们未来将增加转换脚本来实现对 NeuroClaw 全部能力的兼容。
我的数据会被如何处理？
NeuroClaw 默认在本地运行，除非你显式接入外部服务。你的数据、文件和输出由你自己控制；只有当你配置第三方模型后，提示词或中间产物才可能发送到外部后端。
benchmark 主要评测什么？
NeuroBench 评测的是可执行性、输出有效性、规划质量、工具/技能使用是否合理，以及复现准备度。它面向真实神经影像工作流，而不是单轮问答。
benchmark 怎么运行？
可以使用 Web UI：python core/agent/main.py --web --benchmark，也可以使用命令行批量运行：python core/agent/main.py --benchmark。如果要对比技能版和基线版，再加上 --benchmark-compare-skills。
可以跑很长的会话，或者同时开多个会话吗？
可以。NeuroClaw 本身就是为迭代式实验闭环设计的，适合长时间运行。你可以在不同终端或浏览器标签里开多个会话，但实际能力仍取决于本地资源和所选模型后端。
安装向导会配置哪些内容？
内置安装器会引导你配置 Python 环境、相关依赖、神经影像工具链路径、LLM 后端、默认目录以及 Web UI 附件支持。
